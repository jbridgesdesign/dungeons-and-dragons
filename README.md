## 📖 About

This is a SheCodes Bonus Add-on project — a single-page tribute to one of my favorite hobbies: **Dungeons & Dragons**! The page introduces the classic tabletop RPG with images, a brief overview, and links to learn more, while incorporating JavaScript interactivity for the first time. It's live on GitHub Pages!

🔗 **[View the live site](https://jbridgesdesign.github.io/dungeons-and-dragons/)**

---

## ✨ Features

- **Hero Image** — Clickable banner image linking to the D&D Wikipedia article
- **Image Gallery** — A photo collection showcasing D&D gameplay, art, and accessories
- **Image Changer** — JavaScript-powered button to cycle through images dynamically
- **Dark Theme Toggle** — JavaScript-powered light/dark mode switch for the page
- **Wikipedia CTA** — Links out to the full D&D Wikipedia article for readers who want to dive deeper
